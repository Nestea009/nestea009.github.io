HTML y CSS

Cuando se habla de desarrollo web, HTML siempre hace
aparición; es el idioma de programación web por excelencia.
Esto combinado con CSS y a veces JavaScript hace unas webs
increibles, y aunque no sé JavaScript por ahora, voy
a contar un poco mi experiencia con HTML y CSS.

Para empezar quitémonos algunas preguntas de en medio:


¿Son fáciles?

En mi opinión, sí bastante.


¿Tienen una comunidad grande?

Sí, las hay más grandes pero he encontrado mucha
infrormación sobre ellos.


¿Cual es más difícil?

HTML es muy sencillo aunque tenga alguna dificultad, y CSS
es fácil pero hay muchas cosas que tener en cuenta, así
que CSS.



Personalmente he disfrutado mucho programando en estos
idiomas, es sencillo, y se siente bastante satisfactorio acabar
una animación o página y ver que funciona.


El uso de estos idiomas me ha hecho crecer bastante como
programador además de permitirme hacer esta web, gracias
a la cual voy a aprender otros idiomas.


Probablemente los idiomas en los que más he disfrutado
trabajar y en los que menos problemas he tenido,
así que 100% recomendado👍.


-Néstor Blanco 25-09-2022
