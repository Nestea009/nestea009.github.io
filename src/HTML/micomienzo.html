Mi Comienzo

Hoy es día 11-09-2022, y con 16 años,
he estado la última semana tratando de crear una web para
documentar todo mi progreso. Como podrás haber deducido esta es la
web resultante, hecha con HTML y CSS, idiomas que
en un principio no sabía manejar, pero que he ido aprendiendo.
Para mi sorpresa, estos dos idiomas han sido probablemente los
más fáciles que he aprendido hasta la fecha, y aunque
no sepa muchos, me ha costado más aprender Python que estos
dos juntos, no sé si porque son fáciles o porque me he acostumbrado
a aprender programación, pero de cualquier manera me alegro de
que me sea sencillo.


En este blog también quería hablar sobre las otras opciones que consideré
para hacer esta web.
Lo primero que probé fue 000webhostapp
pero me acabó dando demsiados problemas así que decidí probar
otras opciones, como WIX o WordPress. Estas no estaban mal del todo,
pero WIX tenía una marca de agua demasiado grande y visible que arruinaba el
diseño, y no supe muy bien manejar e instalar WordPress.


Antes de estas dos opciones también pensé en hacer una web con html
pero mirando ejemplos de webs hechas SOLO con html me hice a la
idea de que no podías hacer buen diseño con este idioma, lo
cual es cierto, pero no consideré la opción de combinarlo con
CSS para que quedase mejor.


Todo esto ha sido una nueva experiencia para mí, un tanto confusa,
pero que al final está saliendo bien. No me he enfrentado a demasiados
problemas trabajando y ninguno de ellos ha sido nada grande así que
espero que siga así.
El diseño web es algo que se me escapa un poco pero
sabiendo que tengo todo el tiempo que quiera creo que iré
haciendo retoques a la web para que se vea mejor y así
tener una web decente dentro de un tiempo, ya que ahora dudo
con el nombre y el fondo de la web, además de que el apartado
"sobre mi" carece de buen diseño.


Nada de esto me hecha para atrás ya que sé que tengo
tiempo para retocar todo, y que no tengo prisa alguna,
esta web es algo progresivo y que necesita dedicación.
Por ahora estoy contento con como está saliendo todo
y con lo que estoy aprendiendo, así que espero seguir así.

-Néstor Blanco 11-09-2022
