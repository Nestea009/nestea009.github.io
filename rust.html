Rust

Rust es una maravilla de lenguaje, personalmente mi
favorito. Rust cuenta con una sintaxis bastante fácil
de entender pero que mantiene todo el control que otros
lenguajes de programación brindan.

Aparte de esto tiene una rapidez innata, es prácticamente
tan rápido como C o C++. He encontrado que para complementar
esto, Rust cuenta con una variedad de extensiones que
amplian todo esto.


Algunos de los proyectos que he hecho en Rust han sido:
- Calculadoras
- Algoritmos simples
- Páginas Web
- Demás proyectos pequeños


De hecho cuando acabé de aprender Rust quise hacer un
proyecto para interiorizar mejor las bases y ganar
velocidad, pero debido a la poca información que
hay sobre este lenguaje, y las escasas extensiones
que había para lo que quería hace, lo tuve que
hacer en Python finalmente.

-Néstor Blanco 11-06-2023
